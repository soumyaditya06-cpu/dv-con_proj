# Stage 2A: Task-Aware Object Detection Pipeline

###  File Overview

* **`task_guided_selector.py`**: Handles the core logic for selecting the most contextually appropriate object. 
* **`pipeline.py`**: Glues backend.
* **`frontend_ui.py`**: Sets up a basic UI.

###  Running the pipeline:

1) Run the main cell in the .ipynb file (might take up to a few minutes)
2) The gradio UI inside the notebook (OR the link to the gradio webpage) is currently the only way to interact with the pipeline and uploading images.
3) A second cell has been provided to print the list of scores (optional to run)
